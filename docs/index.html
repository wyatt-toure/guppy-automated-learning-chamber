Guppy Learning over WiFi (GLoW) Homepage

Lead: Beatriz Quineche
Supervisors: Wyatt Toure, Raina Fan, Simon Reader

Welcome to the project site for the Guppy Learning over WiFi (GLoW) Experiment. This was an Honour's research project conducted by Beatriz Quineche which aimed to establish automated methods for conducting behavioural experiments in guppies.

Beatriz designed an automatic feeder using a Raspberry Pi 4 and Python programming for the automated conduction of learning experiments in guppies. To validate whether learning was possible with this automated method, Beatriz conducted a learning experiment where fish would learn that a light stimulus predicts feeding time. This first simple training regime was to act as proof-of-concept, i.e. to establish whether we could observe fish learning with this automated set-up. You can see the design of the apparatus, documentation for the Python code required to run the chamber, documentation for the R code to conduct the analysis on the data and sample footage of the learning chamber on this site.
